# keeframes
A social media website for short form editing content

## Technologies
- Database:
    - MySQL
- Backend:
    - Flask
- Frontend:
    - React

## Backend
### How to install packages
```
cd api
poetry install
```
### How to activate environment
#### Windows
```powershell
Invoke-Expression (poetry env activate)
```
#### Linux
```bash
eval $(poetry env activate)
```
